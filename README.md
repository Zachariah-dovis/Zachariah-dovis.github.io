# Zachariah Zhang Personal Website

This is a GitHub Pages-ready personal website.

## Structure

- `index.html` — About-only landing page
- `blog.html` — Blog index page
- `posts/` — migrated local blog posts
- `assets/` — local images from the Notion export
- `styles.css` — site styling
- `script.js` — footer year and blog search
- `.nojekyll` — keeps GitHub Pages serving static files directly

## Deploy

Upload all files and folders to the root of the repository:

```text
Zachariah-dovis.github.io
```

Then open:

```text
https://Zachariah-dovis.github.io
```

If GitHub Pages is not enabled, go to **Settings → Pages**, choose **Deploy from a branch**, select `main` and `/root`, then save.
